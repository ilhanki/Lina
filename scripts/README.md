# Scripts Klasörü

Bu klasör geliştirme ve bakım amaçlı yardımcı script'ler için ayrılmıştır.

Örnek kullanım alanları:

- Kurulum yardımcıları.
- Yerel tanı ve kontrol script'leri.
- Model yönetimi.
- Veritabanı migration işlemleri.
- Bakım görevleri.

Script'ler mümkün olduğunca iş mantığını tekrar yazmamalı, `src/lina` altındaki kodu çağırmalıdır.
