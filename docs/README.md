# Dokümantasyon

Bu klasör Lina'nın proje dokümantasyonunu içerir.

Burada mimari kararlar, yol haritası, geliştirme notları, kurulum rehberleri ve gelecekte kullanıcıya dönük dokümanlar tutulur.

Agent Mode için [görev şablonları](agent-task-templates.md) ve [kurtarma ve güvenilirlik](agent-recovery.md) ayrı referanslardır. Premium masaüstü kabuğunun referansla eşlemesi [uygulama notlarında](reference-ui-implementation.md), sürüm özeti ise [v0.12.2-alpha sürüm notlarında](release-notes-v0.12.2-alpha.md) bulunur.

Dokümantasyon dili Türkçedir. Kod örnekleri, dosya yolları ve API isimleri İngilizce kalır.
